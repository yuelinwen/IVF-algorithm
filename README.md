# Inverted File Index (IVF) 向量相似度搜索

用 Python 从零实现 IVF 算法，面向初学者，帮助理解向量数据库的核心原理。

---

## 什么是 IVF？

**问题背景：** 在 AI 应用中（如语义搜索、推荐系统），我们需要在数百万个高维向量里找到最相似的那个。如果逐一比较，数据量大时会非常慢。

**IVF 的解法：** 先把向量"分区"，搜索时只去最相关的区域，跳过其他区域。

> 类比：在图书馆找一本书，你不会从第一排翻到最后一排，而是先去对应的主题分区，再在那个区里找。

---

## 核心原理：三步走

### 第一步：K-Means 聚类（训练阶段）

把所有向量用 K-Means 算法分成 `k` 个簇（cluster），每个簇有一个**中心点（centroid）**。

```
所有向量
  ↓ K-Means 聚类
簇0（中心: c0）: [v2, v5, v11, ...]
簇1（中心: c1）: [v0, v7, v13, ...]
簇2（中心: c2）: [v1, v4, v19, ...]
...
```

**K-Means 的工作方式：**

1. 从数据里随机抽 k 个向量作为**初始中心**（起点，不是最终结果）
2. 把每个向量分配给离它最近的中心 → 形成 k 个簇
3. 重新计算每个簇的**平均值**作为新中心（中心会移动）
4. 重复步骤 2-3，直到中心不再移动（收敛）

> 关键理解：随机初始化只是起点，中心通过迭代"漂移"到每组数据真正的重心位置。

`k` 在 Faiss 等库中也叫 `nlist`，是同一个参数。

---

### 第二步：建立倒排索引（索引阶段）

记录每个簇里包含哪些向量，存成字典：

```python
inverted_index = {
    0: [(vec_id, vec), (vec_id, vec), ...],   # 簇0 里的所有向量
    1: [(vec_id, vec), (vec_id, vec), ...],   # 簇1 里的所有向量
    2: [(vec_id, vec), (vec_id, vec), ...],   # 簇2 里的所有向量
}
```

叫"倒排"是因为结构是 **簇 → 向量列表**，而不是普通的 **向量 → 值**。

---

### 第三步：IVF 搜索（查询阶段）

搜索分三步，关键是**不扫描全部数据**：

```
查询向量 query
    │
    ├─ 1. 和 k 个中心点比相似度        （只算 k 次，很快）
    │      找出最近的 n_probe 个簇
    │
    ├─ 2. 只进入这 n_probe 个簇        （跳过其他簇）
    │      和簇内向量逐一比相似度
    │
    └─ 3. 返回相似度最高的 top_k 个结果
```

---

## 最核心的权衡：`n_probe`

`n_probe` 控制搜索几个簇，是 IVF 最重要的参数：

| n_probe | 召回率 | 扫描向量数 | 速度 |
|---------|--------|-----------|------|
| 1 | 低（可能漏掉） | 最少 | 最快 |
| 2-3 | 中高 | 适中 | 较快 |
| k（全部） | 100%（等同暴力搜索） | 全部 | 最慢 |

**n_probe 越大 → 结果越准确，但速度越慢。** 实际使用中根据业务对准确率和速度的要求来调整。

## 代码结构

```
src/main.py
├── cosine_similarity()     计算两向量的余弦相似度
├── find_nearest_cluster()  找最近的簇中心
├── kmeans()                K-Means 聚类，输出 k 个中心点
├── build_ivf_index()       建立倒排索引
├── ivf_search()            IVF 搜索（核心）
├── brute_force_search()    暴力搜索（用于对比召回率）
└── main()                  演示：构建索引、搜索、对比结果
```

## IVF 与真实系统的关系

本项目实现的是 IVF 的简化版，真实系统（Faiss、Milvus、Pinecone）基于同样原理，加了更多优化：

| 概念 | 本项目 | Faiss |
|------|--------|-------|
| 簇数量 | `k` | `nlist` |
| 搜索簇数 | `n_probe` | `nprobe` |
| 相似度度量 | 余弦相似度 | L2 / 内积 / 余弦 |
| 初始化策略 | 随机采样 | K-Means++ |
